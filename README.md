**这款 App 是什么来头**

心情罐罐是沂水小守护软件技术工作室开发的一款 iOS 情绪手账应用，App Store 评分 4.4 分（474 个评分），定位是"私人情绪收纳师"。简单说就是：你每天点一下当天心情对应的表情，可以加文字、加图片，时间长了它会帮你画成日历、柱状图、环形图，让你看清自己这一年的情绪轨迹。

它和那种一本正经写日记的 App 不太一样，更像是给情绪找个软乎乎的容器。界面是那种治愈系的小清新风格，没什么压迫感，不像有些 productivity 类 App 打开就想让你"完成任务"。

**核心功能挑几个说说**

表情化记录这件事看着简单，但确实是心情罐罐的灵魂。内置 40 多种表情图案，长按还能换成自定义图片。我有时候心情复杂，就用一张自己存的 meme 图代替，比文字更准。

AI 智能识别心情是我比较意外的功能。你可以直接语音说一段话，它会调用 AI 帮你识别情绪状态，还能润色表达。懒得打字的时候特别好用，对着手机嘟囔两句就记完了。

数据可视化部分做得挺用心。心情日历上每天一个小表情，远远看过去哪个月开心哪个月低落一目了然；柱状图按月汇总，环形图按年/月看占比。它还会帮你找出一年里"最闪耀的月份"，告诉你那个月你过得最开心——这种小细节有种被温柔对待的感觉。

iCloud 多端同步、面容 ID 加锁、批量导出日记、补录过往心情这些就属于"用上了就回不去"的实用功能。尤其是补录，我经常是事后才想起来要记，没这个功能的话数据就断了。
